The Pipeline

How It Works

Three ML disciplines — one unified system. All running on real e-commerce data from the UCI Machine Learning Repository.

Customer Segmentation

Unsupervised learning groups customers into Champions, Loyal, At Risk, and Dormant segments using RFM analysis and clustering.

scikit-learn K-Means RFM

Churn Prediction

Supervised classification identifies customers about to leave. Uses PR-AUC with SMOTE for handling class imbalance properly.

XGBoost LightGBM SMOTE

Lifetime Value Forecasts

Neural network regression predicts how much each customer will spend over the next 6 months with temporal train/test splits.

TensorFlow Keras Deep Learning
